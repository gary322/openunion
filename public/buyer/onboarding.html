Developer reference: API endpoints
  1. Create org + first API key: POST /api/org/register
  2. Verify domain: POST /api/origins then POST /api/origins/:id/check
  3. Per-org CORS allowlist: PUT /api/org/cors-allow-origins
  4. Platform fee + wallet: PUT /api/org/platform-fee
  5. Register app type: POST /api/org/apps
  6. Create + publish: POST /api/bounties, POST /api/bounties/:id/publish
  7. Disputes posture: 1-day hold. After hold, auto-refund runs as refund = gross - Proofwork fee.
